# دليل إضافة الصور للتطبيق

## الملفات المطلوبة

يجب إضافة الصور التالية في مجلد `assets/images/`:

### 1. أيقونة التطبيق (App Icon)
- **الاسم:** `app_icon.png`
- **الحجم الموصى به:** 1024x1024 بكسل
- **الاستخدام:** أيقونة التطبيق عند تثبيته على الجوال
- **الخلفية:** يجب أن تكون شفافة أو بخلفية خضراء (#00A65A)

### 2. لوجو التطبيق (Logo)
- **الاسم:** `logo.png`
- **الحجم الموصى به:** 512x512 بكسل أو أكبر
- **الاستخدام:** اللوجو في صفحات التطبيق (Profile Screen)
- **الخلفية:** شفافة أو بخلفية بيضاء

### 3. لوجو أبيض (White Logo)
- **الاسم:** `logo_white.png`
- **الحجم الموصى به:** 512x512 بكسل أو أكبر
- **الاستخدام:** اللوجو في Splash Screen و HomeScreen (على خلفية خضراء)
- **الخلفية:** شفافة مع لوجو أبيض

## خطوات الإضافة

1. أنشئ مجلد `assets/images/` في جذر المشروع (إذا لم يكن موجوداً)
2. ضع الصور الثلاثة في المجلد
3. تأكد من أن أسماء الملفات مطابقة تماماً:
   - `app_icon.png`
   - `logo.png`
   - `logo_white.png`

## توليد أيقونة التطبيق

بعد إضافة `app_icon.png`، قم بتشغيل الأمر التالي لتوليد أيقونات التطبيق:

```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

أو:

```bash
dart run flutter_launcher_icons
```

## ملاحظات

- إذا لم تكن الصور موجودة، سيتم عرض أيقونات افتراضية
- تأكد من أن الصور بصيغة PNG
- للحصول على أفضل جودة، استخدم صور عالية الدقة (1024x1024 أو أكبر)
